Listas não ordenadas

A tag ul (unordered list - lista não ordenada) representa uma lista de itens sem ordem rígida, por exemplo igredientes em uma receita

A tag li (list itens - Itens da lista) representa um item que faz parte de uma lista

Exemplo de lista

  • primeiro item
  • segundo item
  • Terceiro item
  • subitem

      • Listas ordenadas

        A tag ol (ordered list - lista ordenada ) representa uma lista de itens numerdos sequencialmente

        Exemplo de lista ordenada

        1. primeiro item
        2. segundo item
        Atributos

        reversed (ordem reversa)
        Exibe os itens da lista ordenada em ordem reversa

        1. item mais impotante
        2. item menos importante

        start (inicio)
        específica o valor inicial para a numeração da lista

        1. Item 5
        2. Item 6

        type (tipo)
        indica um tipo de numeração: a - indica letras minúsculas A - indica letras maiúsculas i - indica algarismos romanos minúsculos I - indica algaritimos romanos maiúsculos

        1. primeiro item
        2. segundo item
